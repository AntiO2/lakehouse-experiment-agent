# 数据准备

数据生成分为两个 Benchmark，操作相互独立。

## HyBench

### 生成

仓库：https://github.com/AntiO2/pixels-benchmark

```bash
cd ${PIXELS_BENCHMARK_REPO}
# 具体命令见仓库 README
# 生成后每个表对应一个 CSV 文件
```

### CSV 切分（可选）

生成的是单文件 CSV，可以切分为 `part-00000.csv` ~ `part-NNNNN.csv` 便于并行导入：

```bash
./scripts/00_data_generation/split_csv.sh \
  /path/to/data.csv \
  /path/to/output_dir \
  3000000
```

### 上传到 S3

```bash
./scripts/00_data_generation/sync_csv_to_s3.sh sf100
```

上传后 S3 结构：

```
s3://${S3_BUCKET}/data/hybench/sf100/csv/
├── customer/part-00000.csv ...
├── company/
├── savingaccount/
├── checkingaccount/
├── transfer/
├── checking/
├── loanapps/
└── loantrans/
```

### 规模命名

| 规模 | CSV 路径 | 示例上传命令 |
|------|---------|------------|
| SF1 | `${DATA_HYBENCH_1}` | — |
| SF100 | `${DATA_HYBENCH_100}` | `./sync_csv_to_s3.sh sf100` |
| SF1000 | `${DATA_HYBENCH_1000}` | `./sync_csv_to_s3.sh sf1000` |
| SF1333 | `${DATA_HYBENCH_1333}` | `./sync_csv_to_s3.sh sf1333` |

```bash
# SF1000 专用：切分 + 上传
./scripts/00_data_generation/split_csv.sh \
  ${DATA_HYBENCH_1000}/customer.csv \
  ${DATA_HYBENCH_1000}/customer/ \
  3000000
# 对 8 张表逐一执行 split
./scripts/00_data_generation/sync_csv_to_s3.sh sf1000
```
